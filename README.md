# Sadaru Portfolio Website

A premium cinematic portfolio website for **Sadaru**, built with static HTML, CSS, and JavaScript.

## Features

- Moody editorial design with cinematic dark tones
- Responsive layout for desktop, tablet, and mobile
- Animated scroll reveals
- Smooth navigation with active section highlighting
- Project cards with subtle 3D hover tilt
- Copy-email button
- Social links for YouTube, TikTok, Instagram, and Facebook
- No external dependencies or build step required

## Files

- `index.html` — main website content and structure
- `styles.css` — visual design, layout, animations, responsive styles
- `script.js` — interactions, navigation, reveal animations, tilt effect

## How to Open

Double-click `index.html`, or open it in your browser.

## How to Deploy

You can deploy this site to:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Any static hosting provider

For Netlify or Vercel, upload the folder containing these three files.
